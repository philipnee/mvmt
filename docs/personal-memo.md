# Let's Not Get More Cloud

mvmt came from a problem I kept having. I wanted different AI agents and people to be able to use files, notes, computer programs on my computer, without me constantly copying things around or opening everything up to everyone.

I came up with the name on a bike ride along Page Mill Road, Palo Alto. mvmt means movement. I like it a lot because the whole problem is about movement — context moving between me, my data, other people, and these models.

The goal is simple: keep stuff where it already is, on my machine, and let me control who gets access to what, for how long, and for what reason. Some access should be temporary. Some should be really limited. Some can stick around longer. But it should be my choice.

I never really bought the answer of “just put it in the cloud.”: I don't think we need to add more infrastructure to our life. I’m happy with my stuff living on my computer.
